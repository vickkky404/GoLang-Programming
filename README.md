# <img src="https://go.dev/blog/go-brand/Go-Logo/PNG/Go-Logo_Blue.png" alt="Go Logo" width="40" /> Go Programming Tutorials

A collection of Go (Golang) tutorial programs designed to help you learn the language through simple, practical examples.

---

## 📖 Overview

This repository contains small, focused Go programs that demonstrate core concepts of the Go programming language. Each example is written to be clear, beginner-friendly, and easy to run.

Whether you're just getting started or reviewing the basics, these tutorials provide hands-on practice.

---

## 📚 Topics Covered

- Go syntax and program structure
- Variables and data types
- Constants
- Control flow (`if`, `switch`, loops)
- Functions
- Arrays and slices
- Maps
- Structs and methods
- Interfaces
- Error handling
- Pointers
- Concurrency (goroutines and channels)
- Packages and modules
- File handling

---

## 🛠 Requirements

- Go 1.18 or later (recommended)

Download Go from: https://go.dev/dl/

Verify installation:

```bash
go version
```

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repository-name.git
```

### 2. Navigate to the project directory

```bash
cd your-repository-name
```

### 3. Run a tutorial program

```bash
go run filename.go
```

Or build and execute:

```bash
go build filename.go
./filename
```
